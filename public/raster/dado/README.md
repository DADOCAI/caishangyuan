# 光栅线条效果 - 图像风格化工具

一个基于Web的图像处理工具，可以将普通图片转换为具有光栅线条效果的艺术作品。

## 功能特性

### 📁 图像上传
- 支持 JPG、PNG 格式
- 拖拽上传或点击上传
- 快捷键：`Ctrl+O`

### 🎨 图像预处理
- **模糊**：调节线条边缘的柔化程度 (0-10)
- **颗粒**：多层超强噪声效果 (0-200)
- **Gamma**：调节图像整体亮度对比 (0.1-3.0)
- **黑点/白点**：精确控制图像的明暗范围 (0-255)

### 🎯 光栅线条效果
- **显示效果**：启用/禁用光栅线条效果
- **使用原色**：线条使用原图颜色而非纯黑色
- **阈值**：控制黑白切换点，决定哪些区域显示线条 (0-255)
- **网格类型**：常规网格或弯曲网格
- **网格角度**：旋转线条方向 (0-180°)
- **X/Y 网格数**：控制线条密度 (10-200)
- **最小/最大线宽**：控制线条粗细范围

### 💾 导出功能
- PNG 格式：高质量位图
- SVG 格式：矢量图形
- 快捷键：`Ctrl+S`

## 使用方法

1. **上传图像**
   - 点击"上传图片"按钮或拖拽图片到画布区域
   - 支持的格式：.jpg, .png

2. **调整画布大小**
   - 使用"画布大小"滑块调整输出尺寸 (200-1000px)

3. **预处理图像**
   - 根据需要调整模糊、颗粒、Gamma等参数
   - 实时预览效果变化

4. **配置光栅效果**
   - 勾选"显示效果"来启用/禁用线条效果
   - 勾选"使用原色"让线条保持原图的颜色
   - 调整阈值控制线条的显示范围
   - 设置网格数量和线宽范围

5. **导出结果**
   - 选择输出格式（PNG或SVG）
   - 点击"导出画布"下载处理后的图像

## 效果说明

光栅线条效果的工作原理：
- 将图像分割成网格
- 计算每个网格区域的平均亮度和颜色
- 根据亮度值决定是否绘制竖直线条
- 较暗的区域绘制较粗的线条
- 较亮的区域绘制较细的线条或不绘制
- 可选择使用原图颜色或纯黑色线条

最终效果类似于传统的光栅印刷技术，能够用简单的线条变化表现出丰富的明暗层次。启用原色功能后，还能保持原图的色彩信息，创造更加生动的艺术效果。
